{%- comment -%} Previous/next navigation for a collection sub-page, plus a link back to the collection landing page. Jekyll only populates page.previous/page.next for posts, so the sequence is derived here from the same `order`-sorted list the landing page and the header dropdown use. Parameters: collection — collection name, e.g. "programs" (also gives the // landing page URL) label — singular noun for the prev/next captions, e.g. "Program" all_label — plural noun for the back link, e.g. "Programs" Usage: {% include collection-nav.html collection="programs" label="Program" all_label="Programs" %} {%- endcomment -%} {% assign items = site[include.collection] | sort: "order" %} {% assign current = 0 %} {% for item in items %} {% if item.url == page.url %}{% assign current = forloop.index0 %}{% endif %} {% endfor %} {% assign last = items.size | minus: 1 %}
